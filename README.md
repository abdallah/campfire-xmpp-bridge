campfire-xmpp-bridge
====================

Campfire to XMPP bridge

Requirements: 
-------------
* [PyFire]
* [SleekXMPP]

Usage:
------
```sh
# start using:
python xamppfire.py

# stop using: 
pkill xamppfire.py

```

[PyFire]:https://github.com/mariano/pyfire
[sleekxmpp]:https://github.com/fritzy/SleekXMPP
